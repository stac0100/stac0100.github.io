HTML Notes

Index
img tag
This tag displays an image on the webpage. It requires the src and alt attributes.
form tag
This tag creates a space where information can be collected by the browser. Any information you'd like to collect should be place in between the opening and closing form tags. It requires the action and method attributes.
anchor tag
This tag creates a link on our webpage.The href attribute is used with this tag to provide the location of the link. A reference to the resource can be made using an absolute or relative path.
thead tag
This tag adds a heading row within a table.
table tag
This tag is used to define a container where the elements of the table will be placed.
